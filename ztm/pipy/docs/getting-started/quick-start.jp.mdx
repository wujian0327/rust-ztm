title: "クイックスタート"
---

学習用、開発用、デバッグ用にPipyをローカル環境にインストールすることを推奨します。ソースからビルドするか、ご使用の特定のOS向けのビルド済のリリースをダウンロードしてインストールします。Pipyには管理用のウェブコンソールとドキュメント、チュートリアルが付属しています。

ローカル環境にインストールしたら、引数を付けずに `Pipy` を実行するとポート `6060` で管理コンソールが起動します。また `--admin-port=` 引数を付けると別のポートでリッスンできるように設定できます。

### CLIから実行する

Pipyプロキシを起動するには、PipyJSスクリプトファイルで `pipy` を実行します。例えばリクエストを受けるたびに同じメッセージ本体を応答するシンプルなエコーサーバーが必要であれば、`tutorial/01-hello/main.js` のスクリプトを使用します。

```sh
$ pipy tutorial/01-hello/main.js
```

あるいは開発中やデバッグ中には、PipyをビルトインしたウェブUIで起動できます。

```sh
$ pipy tutorial/01-hello/main.js --admin-port=6060
```

### コマンドラインオプションを表示する

```sh
$ pipy --help
```

### ビルトインフィルターとそのパラメーターをリスト表示する

```sh
$ pipy --list-filters
$ pipy --help-filters	
```
