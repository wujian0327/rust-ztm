## Report a bug 

Once you encounter any bug, you can check the [issue list](https://github.com/flomesh-io/pipy/issues/) first and it may be reported or fixed. If nothing find, you can report via raising [GitHub issue](https://github.com/flomesh-io/pipy/issues) to let us know.

## Contribute

Except for bug, you can contribute new feature/documentation, or raise any suggestion too by following steps in [contributing document](https://github.com/flomesh-io/pipy/blob/main/CONTRIBUTING.md).
