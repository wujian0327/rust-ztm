## 报告问题

一旦遇到任何问题，可以先检查[ issue 列表](https://https://github.com/flomesh-io/pipy/issues/)，因为可能会找到相同的问题或者已经被修复。假如没有找到，可以通过创建 [Github issue](https://github.com/flomesh-io/pipy/issues) 告诉我们。

## 贡献

除了报告问题，同样可以按照[贡献文档](https://github.com/flomesh-io/pipy/blob/main/CONTRIBUTING.md)的说明贡献新功能/文档、或者提出任何建议。
