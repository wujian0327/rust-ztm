Pipy 为网络代理应用场景提供了丰富的 API。

# Configuration

[Configuration](/reference/api/Configuration) 是用于配置管道布局的核心 API。所有的 Pipy 模块都应当是一个求解出 _Configuration_ 对象的 PipyJS 表达式。

使用内置的全局函数 [pipy()](/reference/api/pipy) 来创建一个新的 _Configuration_ 对象。
