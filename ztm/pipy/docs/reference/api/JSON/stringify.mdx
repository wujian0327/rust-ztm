## Description

<Summary/>

This method returns a string for the formatted text. When only the binary data is needed, use [JSON.encode()](/reference/api/XML/encode) instead, which can be more efficient without the need of an extra string conversion.

## Syntax

``` js
JSON.stringify(value)
```
