## Description

<Summary/>

If an entry cannot be found with the specified key, and an _add_ callback was given when calling [new Cache()](/reference/api/algo/Cache/constructor), the callback function will be called with the key to obtain its value, and with that value, a new entry will be created in the **Cache**.

## Syntax

``` js
cache.get(key)
```
