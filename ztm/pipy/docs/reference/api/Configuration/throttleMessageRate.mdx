## Description

<Summary/>

<FilterDiagram
  name="throttleMessageRate"
  input="Event"
  output="Event"
/>

This filter passes down everything unaltered. It doesn't change anything in the event stream. It only delays them.

## Syntax

``` js
pipy()
  .pipeline()
  .throttleMessageRate(
    new algo.Quota(quota, { per })
  )

pipy()
  .pipeline()
  .throttleMessageRate(
    () => getQuota()
  )
```
