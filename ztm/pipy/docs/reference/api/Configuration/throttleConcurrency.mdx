## Description

<Summary/>

<FilterDiagram
  name="throttleConcurrency"
  input="Event"
  output="Event"
/>

This filter passes down everything unaltered. It doesn't change anything in the event stream. It only delays them.

## Syntax

``` js
pipy()
  .pipeline()
  .throttleConcurrency(
    new algo.Quota(quota)
  )

pipy()
  .pipeline()
  .throttleConcurrency(
    () => getQuota()
  )
```
