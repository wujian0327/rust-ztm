## Description

<Summary/>

This method can only be invoked right after adding a _joint-filter_ (See [Sub-pipeline](/intro/concepts#sub-pipeline)) to the _Configuration_. Both sub-pipeline layout names and in-place sub-pipeline layouts are supported.

## Syntax

``` js
pipy()
  .pipeline()
  .someJointFilter().to(subPipelineLayoutName)
  .otherFilters()
  // ...

pipy()
  .pipeline()
  .someJointFilter().to(
    $=>$
    .filterA()
    .filterB()
    // ...
  )
  .otherFilters()
  // ...
```
